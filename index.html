Searching for an Animal Protocol
  • Protocol # - Enter a complete or partial Animal Protocol number then click on Search.
  • PI/CoPI/Technician - Begin searching for personnel included on Animal Protocol(s) by entering a minimum of 3 characters of the last name. The application will begin to search and then display a list of names that match. Select the personnel you are looking for and click Search to see a list of all Animal Protocol(s) where they are named. To limit the search results, continue to enter more characters, entering the last name, first name. i.e. Doe, John or enter the HAWKID for an exact match.
Copy and Start a New Animal Protocol from an Existing Animal Protocol
  • Perform a search to find the Animal Protocol you would like to copy.
  • Click on the protocol # in the first column to view the Animal Protocol Summary page.
  • On the Animal Protocol Summary page, look in the right hand column for the Copy link.
  • Click on the Copy link to copy this Animal Protocol to a new Animal Protocol form.
  • Note: Copying can only be done by the PI and those authorized by the PI to start a new protocol.
